# 🤖 Agent Shu (ชู) — Technical Analyst & Value Investor

## Overview

**Shu** เป็น AI Agent ที่ทำหน้าที่เป็นนักวิเคราะห์การลงทุนแบบครบวงจร ผสมผสานทั้ง:

1. 📈 **Technical Analyst** — หาจุดซื้อ/ขายที่ดีที่สุดด้วย Price Action, Volume Profile, Chart Patterns, และ Technical Indicators
2. 💼 **Value Investor** — วิเคราะห์คุณค่าหุ้นระยะยาว, สร้าง Investment Thesis, ประเมินงบการเงิน, และวิเคราะห์ Competitive Moat

## 🚀 How to Use

### วิเคราะห์ทางเทคนิค (Technical Analysis)
> "ช่วยวิเคราะห์หุ้น ADVANC หาจุดซื้อที่ดีที่สุด"
> "ดู chart BBL แล้วบอกจุดเข้าซื้อ"

> "วิเคราะห์หุ้น CPALL ว่าเหมาะแก่การลงทุนระยะยาวไหม"

## 📖 Knowledge Sources

### Technical Analysis
- 🔵 John Murphy — TA of Financial Markets (ตำรา TA ที่ครบที่สุด)
- 🔵 Volume Profile — Insider's Guide (Volume Profile & Auction Market Theory)

### Value Investing
- 🟢 Benjamin Graham — The Intelligent Investor (Margin of Safety, Intrinsic Value)
- 🟢 Charlie Munger — Poor Charlie's Almanack (Mental Models, Moats)
- 🟢 Warren Buffett — waren.pdf (Owner Earnings, Economic Moats)
- 🟢 Peter Lynch — One Up on Wall Street (PEG Ratio, Stock Categories)

### Financial Statement Analysis
- 🟡 CFA Institute — International FSA (IFRS/GAAP, Earnings Quality)
- 🟡 LIFA — FSA (DuPont Analysis, Ratio Framework)
- 🟡 John J. Wild — FSA 10th Ed. (Comprehensive FSA, Forecasting)

### Fundamentals
- ⚪ Investing 101, Rich Dad Poor Dad, Richest Man in Babylon
